_(`cicd.yml` can be any filename you like)_
--- ### GitHub Actions YAML 
name: CICD
on: push
jobs:
  cicd:

    runs-on: ubuntu-latest
    permissions: { contents: read, packages: write, id-token: write }

    steps:
      # https://github.com/internetarchive/cicd
      - uses: internetarchive/cicd@v1

        with:
          # ACTION ITEM: change values to your nomad cluster
          BASE_DOMAIN: 'example.com'
          NOMAD_ADDR: 'https://nom.example.com'

          # ACTION ITEM: add GitHub Secret to your repository,
          # with the token value for your nomad cluster
          NOMAD_TOKEN: ${{ secrets.NOMAD_TOKEN }}

          # this is automatically taken care of
          REGISTRY_TOKEN: ${{ secrets.GITHUB_TOKEN }}
--- ## GitHub Action - [build] ` - docker build` and `push` to github registry - [test] - if exists, runs `test.sh` - [can customize](https://github.com/internetarchive/hello-js) - [deploy] - runs: [deploy.sh](https://gitlab.com/internetarchive/nomad/-/blob/master/deploy.sh) --- ## Deployment [deploy.sh](https://gitlab.com/internetarchive/nomad/-/blob/master/deploy.sh) Runs inside a container with `nomad` binary
- merges CI/CD Variables & Secrets - into shell environment variables & `env.env` file

--- ## [minimal project](https://gitlab.com/internetarchive/nomad/-/blob/master/etc/hello-world.hcl) --- 
job "hello-world" {
  datacenters = ["dc1"]
  group "hello-world" {
    network {
      port "http" {
        to = 5000
    }}

    task "hello-world" {
      driver = "docker"
      config {
        image = "${var.CI_GITHUB_IMAGE}"
        ports = [ "http" ]
    }}

    service {
      name = "hello-world"
      tags = [
"urlprefix-${var.CI_PROJECT_PATH_SLUG}-${var.CI_COMMIT_REF_SLUG}.${var.BASE_DOMAIN}:443/" ]
      port = "http"

      check {
        type     = "http"
        port     = "http"
        path     = "/"
        interval = "10s"
        timeout  = "2s"
    }}
}}
